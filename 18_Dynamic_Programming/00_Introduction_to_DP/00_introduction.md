# Dynamic Programming (DP)

- DP is a algorithmic technique where we solve a particular sub-problem once, store its result and then reuse it later.
- It is used on overlapping subproblem, where a particular subproblem is repeated.
- To do DP, we can have a iterative approach (Tabulation) or a recursive approach (Memoization), fundamentally both work in a similar fashion.
- Recursion/Memoization is Top-Down, go from required to base case, recursively.
- Tabulation is Bottom-Up, go from base case to the required, iteratively.
