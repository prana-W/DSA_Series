# Arrays

- A data structure that contains data/elements of similar data types (integers, characters, string, pair etc)

- NOTE: If we declare the array globally, it will be initialised with 0s, while in the main, it will be initialised with garbage values
- Max size of an array can be of size 10^6 inside main function, and 10^7 in Global declaration.
- Array's first element address in a random memory address. But using the first element address, we can predict/access the address of other elements, due to address being contiguos
