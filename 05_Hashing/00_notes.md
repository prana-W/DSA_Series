# Map Internal Working

Map Data Structure is internally designed by various methods like:

- Division Method
- Folding Method
- Mid Square Method

NOTE: In map, the key can be of any data structure, data type. In unordered_map, the key can only be the data type and not data structure. 

Example, pair<int, int> can be the key in map but not in an unordered_map
